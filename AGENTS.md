# AGENTS.md

This project uses **[CLAUDE.md](./CLAUDE.md) as the canonical instruction file for AI coding agents.** All rules in CLAUDE.md apply to every AI coding agent working on this repository — Claude Code, Cursor, Codex, Aider, Continue.dev, Gemini CLI, and any others.

If you are an AI agent (or a human reading this on behalf of one):

1. Read [CLAUDE.md](./CLAUDE.md) — the 8 hard rules and "STOP and ask" cases are non-negotiable.
2. Read [CHARTER.md](./CHARTER.md) — scope, architecture, what's IN and OUT.
3. Read [docs/DEV_PROCESS.md](./docs/DEV_PROCESS.md) — the full workflow you must follow.

If you are not Claude Code and your runtime doesn't auto-load `CLAUDE.md`, you are still bound by its rules. Load it manually at the start of every session.
